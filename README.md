## How it works

Everything runs directly in the browser — plain HTML, CSS, and JavaScript. No server, no database, no dependencies to install. Each game is a single self-contained `index.html` file.

## Contributing

New games and contributions are very welcome! 🎉

If you have an idea for a fun browser game or want to improve an existing one, feel free to open a pull request or file an issue on GitHub.

**Adding a new game is straightforward:**

1. Create a folder with your game name (e.g. `my-game/`)
2. Add a single `index.html` file — all HTML, CSS, and JS in one file
3. Use the dark purple background: `background: linear-gradient(135deg, #0f0c29, #302b63, #24243e)`
4. Add a `← Back to Arcade` link at the top pointing to `../index.html`
5. Add a card for your game in the root `index.html` and a row in this README

**Ideas for new games:** Odd One Out, Trivia Quiz, Memory Match, Country Silhouettes, Guess the Year ...

Any skill level welcome — the simpler the better!
